## Developing

To run a local instance of this project:

```bash
git clone https://github.com/edde746/twitter-clone.git
cd twitter-clone
npm install
npm run dev -- --open
```

To run local redis instance:

```bash
git clone https://github.com/redis/redis-om-node.git
cd redis-om-node/docker
docker compose up
```
